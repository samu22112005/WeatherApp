# 🌤 Weather App — Android (Java + Retrofit)

A modern Android Weather App built using **Java, Retrofit, and OpenWeather API**.  
It allows users to check real-time weather by city name or automatically using device location (GPS).

---

## 🚀 Features

- 🔍 Search weather by city name  
- 📍 Get weather using current location (GPS)  
- 🌡 Shows Temperature (°C)  
- 💧 Displays Humidity  
- 🌤 Shows Weather Condition (Clear, Clouds, Rain, etc.)  
- 🎨 Clean and modern UI design  
- ⚡ Fast API response using Retrofit  

---

## 🛠 Tech Stack

- **Language:** Java  
- **UI:** XML (Android Views)  
- **API:** OpenWeather API  
- **Networking:** Retrofit  
- **Location:** Fused Location Provider
